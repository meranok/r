# 📝 To-Do App (Vanilla JavaScript)

Oddiy, yengil va **localStorage** bilan ishlaydigan to-do ilova.  
**Dark/Light mode**, tahrirlash, o‘chirish, bekor qilish va avtomatik saqlash funksiyalari mavjud.

## ✨ Xususiyatlar

- ✅ Yangi vazifa qo‘shish
- ✏️ Vazifani tahrirlash
- ❌ Vazifani o‘chirish
- 🚫 Tahrirlashni bekor qilish (Escape tugmasi yoki Cancel tugmasi)
- 💾 LocalStorage orqali avtomatik saqlash
- 🌗 Dark/Light mode almashtirish va saqlash
- ⚡ Har bir amal real-time yangilanish
